## 🏆 Competition Overview

**Aim:**  
Build a machine learning model that predicts the **presence of heart disease** based on patient clinical and demographic data.

**Dataset:**  
The dataset includes features such as **age, sex, chest pain type, resting blood pressure, cholesterol levels, ECG results, heart rate, exercise-induced angina**, and other medical indicators.

**Problem Type:**  
Binary Classification — evaluated using the **ROC-AUC** metric.

**Citation:**  
> Walter Reade and Elizabeth Park. *Heart Disease Prediction.*  
> https://www.kaggle.com/competitions/playground-series-s6e2, 2025. Kaggle.


## 📒 Notebook Information

**Objective:**  
The goal of this project is to provide a structured and didactic approach to heart disease prediction, explaining each step of preprocessing, feature engineering, and model evaluation.  
The notebook prioritizes interpretability and methodological clarity.

📘 The full notebook is available on Kaggle:  
🔗 https://www.kaggle.com/code/lauraalonso/heart-disease-didactic-notebook

**Structure:**  

* 📑 1. First steps.
* 🔍 2. Exploratory Data Analysis (EDA)
    * a. 📊 Dataset overview.
    * b. 🎯 Target distribution.
    * c. 📈Numerical features analysis.
    * d. 🏷 Categorical features analysis.
    * e. 🔗 Feature associations.
    * f. 📌 EDA conclusions.
* 🛠 3. Feature Engineering
    * a. 📐Train regression models.
    * b. 🔁 Test regression models.
* 🤖 4. Modeling
    * a. 🔹 Logistic Regression (L2 Regularization).
    * b. 🔹 Logistic Regression (ElasticNet).
    * c. 🔹 Random Forest.
    * d. 🔹 LightGBM.
    * e. 🎯 Model Comparison.
    * f. 📌 Models conclusions.
* 🏆 5. Final model
